# Brainstorming 智能规划工具

## 项目概述
基于 [obra/superpowers](https://github.com/obra/superpowers) 的 Brainstorming 技能，构建的苏格拉底式设计提炼互动工具。帮助用户在动手实现之前，通过引导式对话理清思路、探索方案、生成经过验证的设计文档。

## 核心理念（来自 Brainstorming 技能）
- **不立即写代码** — 先退一步搞清楚真正要做什么
- **苏格拉底式对话** — 一次一个问题，逐步深入
- **没有"太简单"的项目** — 每个想法都值得认真设计
- **YAGNI** — 无情地移除不必要的功能
- **探索替代方案** — 始终提出 2-3 种方案再决策
- **增量验证** — 呈现设计后获得用户批准再推进

## 五步流程
1. **输入想法** — 描述你的项目、目标或问题
2. **深入探索** — 智能分析 + 逐个问题澄清（多选题 + 开放式交替）
3. **方案对比** — 生成 2-3 种替代方案，含优劣势分析和推荐
4. **设计确认** — 预览设计文档，完成自检清单，用户审批签字
5. **生成文档** — 导出 Markdown 设计文档，进入实施规划阶段

## 技术方案
- 纯静态 HTML + CSS + JavaScript，零依赖
- 单文件部署，直接浏览器打开即可使用
- 不存储任何过往数据，每次使用都是全新会话

## 功能特点
- 智能领域识别（18 个行业领域）
- 技术栈自动检测（20+ 技术关键词）
- 多选题 + 开放式问题交替引导
- 根据用户回答动态推荐方案
- 设计文档自动生成与自检
- Markdown 导出 & 剪贴板复制
